# Customer-Support-AI
Overview
This project is an AI-powered support ticket analysis system built using Python, FastAPI, Pandas, and Ollama Llama 3.2.

The application allows users to query customer support ticket data using natural language, detect anomalies in ticket records, and access insights through REST API endpoints.

The LLM is used for understanding the user's natural-language question and converting it into a structured intent. The actual data processing, filtering, grouping, aggregation, and anomaly detection are handled using Python and Pandas.
